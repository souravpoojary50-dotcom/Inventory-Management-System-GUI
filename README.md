# Inventory-Management-System-GUI
Inventory Management System (Java GUI)

📌 Project Description

This is a simple Inventory Management System developed using Java and Swing GUI.
It allows users to manage inventory items through a graphical interface.

🚀 Features

- Add new items
- View all items
- Delete items
- User-friendly GUI interface

🛠 Technologies Used

- Java
- Swing (GUI)
- ArrayList

🧠 Concepts Learned

- Object-Oriented Programming (OOP)
- Event Handling
- GUI Development
- CRUD Operations

▶ How to Run

1. Compile the program

javac InventoryManagementSystem.java

2. Run the program

java InventoryManagementSystem

📷 Output

A GUI window will open where you can:

- Enter item details
- Add, view, and delete items
